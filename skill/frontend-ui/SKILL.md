# Skill — Frontend UI/UX

**Spec:** [`spec/002-frontend-ui-ux.md`](../../spec/002-frontend-ui-ux.md)

## Reglas

- Solo frontend en `app/public/` — sin lógica backend
- Seguir variables en `assets/css/variables.css`
- Componentes reutilizables en `components/`
- Jessica lidera; Verenice integra Blade después

## Orden

1. variables.css + style.css
2. layout (navbar, sidebar, footer)
3. login.php
4. dashboard.php
5. productos, categorias, movimientos
6. busqueda.php (Fase 6)
7. ventas.php, compras.php
8. usuarios.php, reportes.php
9. responsive.css + pruebas móvil
